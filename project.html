Color Flipper

This project required the use of HTML, CSS, and Javascript in order to be completed.
There is two HTML files present: one for the hex codes and one for simple colors.
For the simple page, An array was used to contain the colors in a list that was activated by a button.
A getRandomNumber() function was used to select a random color based on the array length (colors.length)
For the hex code page, a array was composed of the numbers of 0-9 and A-F.
A for statement was used that stated that i should start at 0, if i is less than 6, i++ should be added to i.
The getRandomNumber() was used and the hex.length was utilized to pull random numbers and letters until 6 characters were made.
A mistakle that was made is the click event listener was not solely exclusive to the button in the simple color page.
